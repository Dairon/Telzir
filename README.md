# Telzir

Telzir is an application that calculates the cost o phone call.

## Ruby Version

3.0.0

## Rails Version

6.1.3

## Installation

```bash
sudo apt install rbenv
```
```bash
rbenv install 3.0.0
```
```bash
rbenv global 3.0.0
```
```bash
 gem install bundler
```
```bash
gem install rails -v 6.1.3
```
```bash
rbenv rehash
```


## Usage

```bash
rails db:migrate
```
```bash
rails db:seed
```
```bash
bundle install
```
```bash
npm install
```
```bash
rails server
```
